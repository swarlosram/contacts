# 📒 Contact Book

A simple contact book built with JavaScript! This project was built to practice arrays and objects — two of the most important concepts in JavaScript.

## 🌐 How It Works

The contact book stores multiple contacts using an array of objects. Each contact has a name, phone number and email address. The app loops through all contacts and displays them in the console.

## 🛠️ Built With

- JavaScript
- HTML5

## ✨ Concepts Used

- Arrays
- Objects
- Array methods (forEach)
- Dot notation to access object properties
- Template literals to display contact info

## 📸 How to Use

1. Open the project in your browser
2. Open DevTools console (Command + Option + I)
3. See all contacts displayed with their name, phone and email!
